# Как достать номенклатуру и примеры данных из 1С

Ниже запросы, которые можно выполнить в консоли запросов 1С или в обработке "Консоль запросов".

## 1. Справочник продукции (асПродукция)

```bsl
ВЫБРАТЬ
    асПродукция.Код КАК Код,
    асПродукция.Наименование КАК Наименование,
    асПродукция.Артикул КАК Артикул,
    асПродукция.ВидСечения КАК ВидСечения,
    асПродукция.Родитель.Наименование КАК Группа
ИЗ
    Справочник.асПродукция КАК асПродукция
ГДЕ
    НЕ асПродукция.ЭтоГруппа
    И НЕ асПродукция.ПометкаУдаления
УПОРЯДОЧИТЬ ПО
    асПродукция.Наименование
```

Поле `Артикул` чаще всего используется в XML-элементе `<article>`.

## 2. Номенклатура для товаров

```bsl
ВЫБРАТЬ ПЕРВЫЕ 50
    Номенклатура.Код КАК Код,
    Номенклатура.Артикул КАК Артикул,
    Номенклатура.Наименование КАК Наименование,
    Номенклатура.НаименованиеПолное КАК НаименованиеПолное,
    Номенклатура.ЕдиницаИзмерения.Наименование КАК ЕдиницаИзмерения
ИЗ
    Справочник.Номенклатура КАК Номенклатура
ГДЕ
    НЕ Номенклатура.ЭтоГруппа
    И НЕ Номенклатура.ПометкаУдаления
УПОРЯДОЧИТЬ ПО
    Номенклатура.Наименование
```

Для товаров в XML используйте `Артикул` → `<article>`, а `Наименование` → `<name>`.

## 3. Материалы (асМатериалы)

```bsl
ВЫБРАТЬ
    асМатериалы.Код КАК Код,
    асМатериалы.Наименование КАК Наименование,
    асМатериалы.d1 КАК d1,
    асМатериалы.d2 КАК d2,
    асМатериалы.s1 КАК s1,
    асМатериалы.s2 КАК s2,
    асМатериалы.t КАК t,
    асМатериалы.Родитель.Наименование КАК Группа
ИЗ
    Справочник.асМатериалы КАК асМатериалы
ГДЕ
    НЕ асМатериалы.ЭтоГруппа
    И НЕ асМатериалы.ПометкаУдаления
УПОРЯДОЧИТЬ ПО
    асМатериалы.Наименование
```

Поле `t` — это толщина, которая передаётся в `<thickness>`.  
Поле `Код` или собственный идентификатор материала — в `<material>`.

## 4. Соединения воздуховодов (асСоединениеВоздуховодов)

```bsl
ВЫБРАТЬ
    асСоединениеВоздуховодов.Код КАК Код,
    асСоединениеВоздуховодов.Наименование КАК Наименование,
    асСоединениеВоздуховодов.ВидСечения КАК ВидСечения,
    асСоединениеВоздуховодов.НомерПоПорядку КАК НомерПоПорядку
ИЗ
    Справочник.асСоединениеВоздуховодов КАК асСоединениеВоздуховодов
ГДЕ
    НЕ асСоединениеВоздуховодов.ПометкаУдаления
УПОРЯДОЧИТЬ ПО
    асСоединениеВоздуховодов.Код
```

Из модуля известны предопределённые коды:

| Код | Наименование |
|-----|--------------|
| 000000001 | Бандаж |
| 000000002 | Без соединения (круглое) |
| 000000003 | Ниппель |
| 000000004 | Фланец круглый |
| 000000005 | Фланец прямоугольный |
| 000000006 | Шина уголок |
| 000000007 | Рейка |
| 000000008 | Без соединения (прямоугольное) |

> В XML передаётся только **первый символ** значения (`Лев(значение, 1)`), после чего модуль ищет соединение через `асОбщийМодуль.НахождениеСоединениеПродуктаПо_Id_c` / `_Id_r`. Уточните у разработчика/в справочнике, какие именно ID используются в вашей базе.

## 5. Посмотреть уже загруженные/сформированные заказы

```bsl
ВЫБРАТЬ ПЕРВЫЕ 10
    асВводНовогоЗаказа.Ссылка КАК Ссылка,
    асВводНовогоЗаказа.Номер КАК Номер,
    асВводНовогоЗаказа.Дата КАК Дата,
    асВводНовогоЗаказа.Контрагент КАК Контрагент,
    асВводНовогоЗаказа.Заказы.(
        НомерСтроки КАК НомерСтроки,
        Продукт.Наименование КАК Продукт,
        Артикул_Продукта КАК Артикул,
        НаименованиеССайта КАК НаименованиеССайта,
        t КАК Толщина,
        n КАК Количество,
        Артикул_Материала КАК АртикулМатериала,
        СоединениеВоздуховода_c0.Наименование КАК Соединение_c0,
        СоединениеВоздуховода_r0.Наименование КАК Соединение_r0
    ) КАК Заказы
ИЗ
    Документ.асВводНовогоЗаказа КАК асВводНовогоЗаказа
ГДЕ
    асВводНовогоЗаказа.Проведен
УПОРЯДОЧИТЬ ПО
    асВводНовогоЗаказа.Дата УБЫВ
```

Это позволит увидеть реальные примеры заполнения строк.

## 6. Пример заполнения XML на основе реального заказа

Выполните запрос выше (п. 5), возьмите значения из строки и подставьте в шаблон:

```xml
<productRow>
    <article>2-1-1</article>                       <!-- Артикул_Продукта -->
    <name>Воздуховод круглого сечения...</name>      <!-- НаименованиеССайта -->
    <characteristic>D=100 t=0.5</characteristic>     <!-- Характеристика_Продукта -->
    <quantity>5</quantity>                          <!-- n -->
    <material>ST-05</material>                      <!-- Артикул_Материала -->
    <thickness>0.5</thickness>                      <!-- t (всегда число!) -->
    <connection_0>3</connection_0>                  <!-- ID соединения c0/r0 -->
    <connection_1>2</connection_1>                  <!-- ID соединения c1/r1 -->
    <connection_2>0</connection_2>
    <connection_3>0</connection_3>
    <system>Система 1</system>                      <!-- асСистема -->
    <comment>Цвет RAL 9016</comment>                <!-- КомментарийКПродукцииИзOnLine -->
</productRow>
```

## 7. Альтернатива: сгенерировать XML прямо из 1С

Если в базе уже есть корректно заполненные заказы, можно написать небольшую обработку, которая выгрузит их в XML в нужном формате. См. метод `СоздатьXMLСервер` в модуле объекта обработки `асВводНовогоЗаказа`.

Если доступа к объекту нет, можно собрать XML вручную через `ЗаписьXML`:

```bsl
ЗаписьXML = Новый ЗаписьXML;
ЗаписьXML.ОткрытьФайл("C:\\Vozd_Obmen.xml");
ЗаписьXML.ЗаписатьОбъявлениеXML();
ЗаписьXML.ЗаписатьНачалоЭлемента("Order");

ЗаписьXML.ЗаписатьНачалоЭлемента("numberDate");
ЗаписьXML.ЗаписатьТекст(Формат(ТекущаяДата(), "ДФ=dd.MM.yyyy"));
ЗаписьXML.ЗаписатьКонецЭлемента();

// ... остальные поля шапки

ЗаписьXML.ЗаписатьНачалоЭлемента("Tabl");
Для Каждого Стр Из Заказы Цикл
    ЗаписьXML.ЗаписатьНачалоЭлемента("productRow");
    
    ЗаписьXML.ЗаписатьНачалоЭлемента("article");
    ЗаписьXML.ЗаписатьТекст(Стр.Артикул_Продукта);
    ЗаписьXML.ЗаписатьКонецЭлемента();
    
    ЗаписьXML.ЗаписатьНачалоЭлемента("name");
    ЗаписьXML.ЗаписатьТекст(Стр.НаименованиеССайта);
    ЗаписьXML.ЗаписатьКонецЭлемента();
    
    ЗаписьXML.ЗаписатьНачалоЭлемента("thickness");
    ЗаписьXML.ЗаписатьТекст(Формат(Стр.t, "ЧРД=.; ЧН=0; ЧГ=0"));
    ЗаписьXML.ЗаписатьКонецЭлемента();
    
    // ... остальные поля строки
    
    ЗаписьXML.ЗаписатьКонецЭлемента(); // productRow
КонецЦикла;
ЗаписьXML.ЗаписатьКонецЭлемента(); // Tabl
ЗаписьXML.ЗаписатьКонецЭлемента(); // Order
ЗаписьXML.Закрыть();
```

Обратите внимание: `Формат(Стр.t, "ЧРД=.; ЧН=0; ЧГ=0")` гарантирует, что толщина будет числом с точкой в качестве разделителя и `0` вместо пустого значения.
