# Руководство по запуску веб-приложения «PDF-спецификация → XML для 1С»

## Требования

- Windows 10/11
- Python 3.12 (рекомендуется)

## Подготовка окружения

1. Откройте командную строку (cmd) или PowerShell в корне проекта `D:\project`.
2. Создайте виртуальное окружение:

   ```cmd
   python -m venv venv
   ```

3. Активируйте окружение:

   ```cmd
   venv\Scripts\activate
   ```

4. Установите зависимости:

   ```cmd
   python -m pip install --upgrade pip
   python -m pip install -r requirements.txt
   ```

## Запуск приложения

### Вариант 1 — через пакетный файл

Дважды щёлкните `run_web_app.bat` в проводнике или выполните в командной строке:

```cmd
run_web_app.bat
```

### Вариант 2 — вручную

```cmd
venv\Scripts\activate
streamlit run web_app.py
```

После запуска в терминале появится адрес локального сервера (обычно `http://localhost:8501`). Откройте его в браузере.

## Работа с приложением

### 1. Загрузка файла

- Нажмите кнопку загрузки файла.
- Поддерживаются форматы: **PDF**, **CSV**, **XLSX**.
- Для CSV используется разделитель `;` с кодировкой UTF-8.

### 2. Выбор страниц (только для PDF)

- После загрузки PDF отобразится список страниц.
- Выберите нужные страницы через мультиселект (по умолчанию выбраны все).

### 3. Предпросмотр таблиц

- Приложение покажет таблицы, найденные на выбранных страницах.
- Выберите таблицы, которые нужно включить в заказ.

### 4. Редактирование объединённой таблицы

- Объединённые данные откроются в редакторе таблицы.
- Исправьте наименования, размеры, количество, единицы измерения, материал и толщину.
- Убедитесь, что столбцы называются: `name`, `size`, `unit`, `quantity`, `material`, `thickness`.

### 5. Генерация XML

- Нажмите кнопку **«Сгенерировать XML»**.
- Приложение вызовет `process_specification_table.process_rows(...)` и сформирует XML для загрузки в 1С.

### 6. Скачивание результатов

- Скачайте файл `order.xml` для загрузки в 1С.
- Скачайте файл `skipped.json`, чтобы посмотреть строки, которые не удалось распознать автоматически.

## Ограничения

- Круглые диффузоры `ДПУ-М` не загружаются автоматически (нет артикула в каталоге `асПродукция`).
- Агрегатные строки фасонных изделий (`м²`, пустой размер) требуют ручной детализации.
- Точность распознавания таблиц из PDF зависит от качества исходного документа.

## Устранение неполадок

- Если Streamlit не найден, проверьте, что виртуальное окружение активировано.
- Если CSV отображается некорректно, проверьте кодировку и разделитель.
- При ошибках генерации XML проверьте, что все обязательные столбцы (`name`, `size`, `unit`, `quantity`) заполнены.
