## 3. Сверка: PDF vs КП

- **PDF и КП — это разные части проекта.**
  - PDF — это ведомость оборудования **приточных установок** (модули LITENED).
  - КП — это **воздуховоды, арматура и решётки/диффузоры** здания.
- Пересечение по номенклатуре минимально:
  - В PDF есть заслонки `CHR` (маппятся на дроссели 16-1-1/16-2-1).
  - В КП есть противопожарные клапаны, воздушные клапаны, решётки и диффузоры — этих артикулов в `асПродукция` сейчас нет.

## 5. Что нужно доработать

### 5.1. Добавить (или подтвердить) артикулы в 1С / маппинг
Без этих артикулов автозагрузка невозможна:
- Противопожарные клапаны (`ПРОК`, `КПП 90ЭМ220`, `КПП 90МНЕ220`).
- Воздушные/регулирующие клапаны (`КВК-Р`, `АВК-Р`).
- Решётки (`Р-Гс`).
- Диффузоры/анемостаты (`ДПУ`, `Р-ПС`).
- Крышный зонт.
- Клапан избыточного давления КИД.
- Огнезащитная изоляция (скорее всё, закупная позиция).

### 5.2. Доработать парсер размеров
- `600x600-300/Ф200-50` — размеры КСД-адаптера (два сечения + патрубок).
- `КПП 90ЭМ220-НО 150-150` — размеры внутри модельной строки.
- `Р-Гс 150*150` / `ДПУ-100` — размеры внутри наименования, а не в колонке «размер».

### 5.3. Доработать загрузку xlsx в веб-приложении
- Пропускать шапку КП (первые 4 строки).
- Распознавать материал и толщину из одной колонки (например, `оцинк. ст.(08пс)0.70`).
- Убирать служебные строки («Итого», «Вопросы», пустые).
- Приводить количество к числу (из формата `98,000`).

### 5.4. Уточнить выбор артикула для круглых воздуховодов
- Сейчас всё круглое маппится на `1-1-2` (спирально-навивной). Нужно различать:
  - `спирально-навивной` → `1-1-2`
  - `прямошовный` → `1-1-1`

### 5.5. Объединение источников
- В итоговом заказе нужно суммировать PDF-оборудование **и** КП-воздуховоды/арматуру.
