# PDF/CSV/XLSX спецификация → XML для 1С

Инструмент для преобразования спецификаций вентиляционного оборудования (PDF, CSV, Excel) в XML-заказ для загрузки в 1С:КА.

## Запуск веб-приложения

```bash
venv\Scripts\activate
streamlit run web_app.py
```

Или просто запустить `run_web_app.bat`.

## Основные модули

- `web_app.py` — интерфейс Streamlit.
- `process_specification_table.py` — разбор строк спецификации и валидация.
- `generate_order_xml.py` — генерация XML по артикулам и параметрам.
- `pdf_spec_extractor.py` — извлечение таблиц из PDF.
- `equipment_pdf_extractor.py` — извлечение ведомостей оборудования из PDF.
- `map_customer_equipment.py` — сопоставление оборудования с артикулами 1С.
- `match_description_to_product.py` — поиск артикула по текстовому описанию.

## Структура репозитория

- `docs/` — документация и исходные требования.
- `examples/` — примеры входных файлов от заказчиков.
- `reference/` — выгрузки/справочники из 1С.
- `tools/` — вспомогательные файлы (например, обработка 1С).
- `_to_delete/` — временные тестовые генерации (не коммитится).
